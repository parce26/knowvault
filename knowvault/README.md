# KnowVault

KnowVault is an enterprise knowledge management system that allows organizations
to store internal documents and query them using an AI assistant based only on
the uploaded content.

## Main Features
- User authentication
- Upload and manage PDF documents
- Categorization of documents
- AI-powered question answering (RAG)
- Query history

## Tech Stack
- Java
- Spring Boot
- Thymeleaf
- SQL Database (to be integrated)
- AI API

## Project Status
Initial setup completed. Architecture and tasks organization in progress.
